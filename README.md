# 명지대학교 졸업 도우미
### 프로젝트 소개
* 입학연도, 전공별로 각기 다른 졸업 조건을 한 눈에 확인할 수 있습니다.
* 추가로 들어야하는 학점 및 필수 과목을 확인할 수 있습니다.
